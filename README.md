# منصة التعلم التفاعلي - ألعاب التوعية الصحية 🎓

## نظرة عامة
منصة تعليمية تفاعلية متطورة مصممة لتعزيز الوعي الصحي ومكافحة السمنة من خلال الألعاب التفاعلية والفيديوهات التعليمية. تم تطوير المشروع ليكون تطبيق ويب تقدمي (PWA) مع دعم العمل دون اتصال وأداء محسن.

## الميزات التقنية الجديدة ✨

### 1. تطبيق ويب تقدمي (PWA) 📱
- **Service Worker**: للعمل دون اتصال وتحسين الأداء
- **Manifest.json**: لتثبيت التطبيق على الأجهزة
- **صفحة Offline**: تجربة مستخدم محسنة عند انقطاع الاتصال
- **تحديثات تلقائية**: إشعارات عند توفر تحديثات جديدة

### 2. نظام إدارة مركزي 🔧
- **config.js**: إعدادات موحدة للمشروع
- **utils.js**: مكتبة أدوات مساعدة شاملة
- **app.js**: مدير التطبيق الرئيسي
- **معالجة أخطاء متقدمة**: تسجيل وإدارة الأخطاء

### 3. تحسينات الأداء ⚡
- **تحميل كسول**: للصور والموارد الثقيلة
- **تحسينات للأجهزة المحمولة**: أداء محسن للهواتف
- **ضغط الموارد**: تقليل أحجام الملفات
- **تخزين مؤقت ذكي**: استراتيجيات تخزين متقدمة

### 4. تقسيم الألعاب 🎮
- **صفحات منفصلة**: كل لعبة لها رابط مستقل
- **فهرس شامل**: صفحة تجميع جميع الألعاب
- **روابط عميقة**: إمكانية الوصول المباشر لأي لعبة
- **قائمة منسدلة**: تنقل محسن في شريط التنقل

### 5. نظام اختبارات شامل 🧪
- **اختبارات تلقائية**: فحص جودة الكود
- **تقارير مفصلة**: نتائج الاختبارات بصيغة HTML
- **مراقبة الأداء**: قياس سرعة التحميل واستهلاك الذاكرة
- **اختبارات التوافق**: فحص دعم المتصفحات

### 6. واجهة مستخدم محسنة 🎨
- **وضع مظلم متقدم**: تبديل سلس مع حفظ التفضيلات
- **تصميم متجاوب**: يعمل على جميع أحجام الشاشات
- **رسوم متحركة محسنة**: تأثيرات بصرية جذابة
- **إشعارات ذكية**: رسائل تفاعلية للمستخدم

## كيفية الاستخدام 🚀

### 1. تشغيل الموقع
```bash
# افتح الملف الرئيسي
index.html
```

### 2. التنقل المحسن
- **الصفحة الرئيسية**: `index.html` → يوجه إلى `pages/main/home.html`
- **فهرس الألعاب**: `pages/main/games/index.html`
- **الألعاب الفردية**: `pages/main/games/[game-name].html`
- **جميع الألعاب**: `pages/main/games.html`
- **الفيديوهات**: `pages/videos/videos.html`
- **التقييم**: `pages/feedback/feedback.html`

### 3. تشغيل الاختبارات 🧪
```javascript
// في وحدة تحكم المتصفح
runTests(); // تشغيل جميع الاختبارات

// أو تلقائياً في وضع التطوير
CONFIG.DEBUG.ENABLED = true;
```

### 4. تفعيل وضع التطوير
```javascript
// في ملف config.js
CONFIG.DEBUG.ENABLED = true;
CONFIG.DEBUG.LOG_LEVEL = 'debug';
CONFIG.DEBUG.SHOW_PERFORMANCE = true;
```

### 3. إضافة محتوى جديد
- **مشاهد جديدة للطفل**: أضفها في `pages/child/`
- **مشاهد جديدة للأم**: أضفها في `pages/mom/`
- **أصول جديدة**: أضفها في `assets/` حسب النوع

## الدعم والمساعدة 💬

### استكشاف الأخطاء وإصلاحها
1. **تشغيل الاختبارات**: `runTests()` في وحدة تحكم المتصفح
2. **فحص وحدة التحكم**: للرسائل والأخطاء
3. **تفعيل وضع التطوير**: `CONFIG.DEBUG.ENABLED = true`
4. **مراجعة Service Worker**: في أدوات المطور → Application

### 🚀 حل مشكلة اللاج (تم الحل!)
**المشكلة**: لاج عند التنقل بين الألعاب
**الحل**: نظام تنظيف شامل للموارد

```javascript
// أدوات مراقبة الأداء
gameManager.getPerformanceStats();   // إحصائيات الأداء
gameManager.cleanup();               // تنظيف يدوي
navStats();                          // حالة التنقل
forceCleanup();                      // تنظيف إجباري

// إذا واجهت لاج
app.clearAllData();                  // مسح البيانات
location.reload(true);               // إعادة تحميل
```

**راجع**: `PERFORMANCE_GUIDE.md` للتفاصيل الكاملة
