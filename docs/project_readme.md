## 🚀 خطوات الإعداد

### 1. إعداد استضافة الملفات

#### الخيار أ: الاستضافة المحلية
- ضع جميع الملفات في مجلد واحد
- استخدم خادم محلي مثل:
  ```bash
  # Python 3
  python -m http.server 8000
  
  # Python 2
  python -m SimpleHTTPServer 8000
  
  # Node.js
  npx http-server
  ```

#### الخيار ب: الاستضافة المجانية
- **GitHub Pages**: أرفع المشروع على GitHub وفعل Pages
- **Netlify**: اسحب وأسقط المجلد أو اربطه بـ GitHub
- **Vercel**: نفس طريقة Netlify

### 2. إعداد Google Forms للاستبيانات

#### إنشاء الاستبيان الأولي:
1. اذهب إلى [Google Forms](https://forms.google.com/)
2. أنشئ نموذج جديد بعنوان "الاستبيان الأولي - منصة التعلم التفاعلي"
3. أضف الأسئلة المطلوبة مثل:
   - المعلومات الأساسية (العمر، الجنس، مستوى التعليم)
   - الخبرة السابقة مع التعلم التفاعلي
   - التوقعات من التجربة

#### إنشاء الاستبيان النهائي:
1. أنشئ نموذج آخر بعنوان "التقييم النهائي - منصة التعلم التفاعلي"
2. أضف أسئلة التقييم مثل:
   - تقييم المحتوى والألعاب
   - سهولة الاستخدام
   - مدى الفائدة المحققة
   - اقتراحات للتحسين

#### ربط النماذج بالموقع:
1. في كل نموذج، اضغط "إرسال" ← "تضمين HTML"
2. انسخ الكود
3. استبدل `YOUR_FORM_ID` في الملفات HTML بالرابط الفعلي

### 3. إعداد فيديوهات Google Drive

#### رفع الفيديوهات:
1. أنشئ مجلد في Google Drive باسم "فيديوهات المشروع"
2. أرفع فيديوهات قسم الأم (8 فيديوهات)
3. أرفع فيديوهات قسم الطفل (6 فيديوهات)

#### الحصول على روابط التضمين:
1. لكل فيديو: انقر بالزر الأيمن ← "مشاركة"
2. اضبط الإذن على "أي شخص لديه الرابط يمكنه المشاهدة"
3. انسخ ID الفيديو من الرابط:
   ```
   https://drive.google.com/file/d/[FILE_ID]/view
   ```
4. استخدم الصيغة التالية للتضمين:
   ```html
   https://drive.google.com/file/d/[FILE_ID]/preview
   ```

#### تحديث صفحات المشاهد:
في كل ملف مشهد (mom-scene1.html, child-scene1.html, إلخ):
```html
<iframe class="video-iframe" 
        src="https://drive.google.com/file/d/YOUR_ACTUAL_VIDEO_ID/preview" 
        allow="autoplay">
</iframe>
```

### 4. إنشاء باقي صفحات المشاهد

استناداً إلى المثال `mom-scene1.html`، أنشئ:

#### مشاهد قسم الأم:
- `mom-scene2.html` - التواصل الفعال
- `mom-scene3.html` - أهمية القراءة
- `mom-scene4.html` - تنمية الإبداع
- `mom-scene5.html` - البيئة التعليمية
- `mom-scene6.html` - حل المشكلات السلوكية
- `mom-scene7.html` - التعاون مع المدرسة
- `mom-scene8.html` - الخلاصة والتطبيق

#### مشاهد قسم الطفل:
- `child-scene1.html` - عالم الألوان
- `child-scene2.html` - الأرقام المرحة
- `child-scene3.html` - حروفي الجميلة
- `child-scene4.html` - الأناشيد التعليمية
- `child-scene5.html` - عالم الحيوانات
- `child-scene6.html` - النجم الصغير

### 5. تخصيص المحتوى

#### تعديل النصوص:
- غير العناوين والأوصاف في كل صفحة
- حدث مدة المشاهدة لكل فيديو
- أضف النقاط الرئيسية لكل مشهد

#### تحديث التنقل:
- تأكد من صحة روابط التنقل بين الصفحات
- اضبط شريط التقدم في كل مشهد
